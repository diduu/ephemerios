A Simple Temporary Messaging App

Want to start right away ?
Click the button below to open a new chat or join an existing one, by typing its url.

Why Ephemeros ?

ἐφήμερος • (ephḗmeros) m: living only one day; short-lived

This service was created to provide access to a quick and secure way of communicating without leaving any traces.
You should use ephemeros because:

  1. It respects your privacy. All communications are encrypted locally with PGP. Only the people you shared the webchat url knows the informations you are sharing.
  2. It is temporary. No one can read your converstaion once you and your correspondent close the window. No conversations are ever saved by the host.
  3. It is completely open source. You are free to make your clone of the website, host it anywhere you want (tor ?) or read the source code to make sure it is secure. Check out the GitHub repository.
  4. You can use it without limit. (even if you're making a thousand web chats).
